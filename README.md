# clone-momentum
크롬앱(momentum) 클론코딩.<br>
그날 그날의 할일 목록을 관리하기 위한 앱입니다.

### 주요 기능
- [x] 현재 시간 확인 가능
- [x] 현 위치, 온도 확인 가능
- [x] 사용자 이름 입력
- [x] 할 일 목록 추가/삭제
- [x] 새로고침 할때마다 배경 변경
- [ ] 반응형 레이아웃

### 사용 기술
 - language : javascript, css, html
 - api : openWeatherMap, unsplash image
 - framework : jquery
